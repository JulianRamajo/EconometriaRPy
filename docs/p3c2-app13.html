Aplicación 3.13 (Modelos econométricos para datos de panel): Productividad de la industria química en China

En esta aplicación se estimará una función de producción del tipo Cobb-Douglas para una muestra de empresas del sector industrial químico de China. La ecuación de regresión asociada a dicha función es la siguiente:

\[log(Y_{it}) = \beta_1 + \beta_2 log(K_{it}) + \beta_3 log(L_{it}) + \beta_4 log(M_{it}) + e_{it}\]

siendo \(Y\) las ventas totales de cada empresa, \(K\) el stock de capital físico, \(L\) el número de empleados, y \(M\) la cantidad de material usado para fabricar los productos químicos.
