Aplicación 1.1.b (Gestión y representación gráfica de datos): Gestión de datos con R y Python

En esta aplicación se usarán las librerías estándar de R y Python en “ciencia de datos” (data science), replicando los resultados con los correspondientes comandos en ambos lenguajes. En concreto, para el lenguaje R se utilizará la colección de librerías contenidas en el paquete Tidyverse (https://www.tidyverse.org/packages/), y para Python se usarán las librerías NumPy (https://numpy.org/), pandas (https://pandas.pydata.org/), Matplotlib (https://matplotlib.org/) y seaborn (https://seaborn.pydata.org/). Las librerías especializadas en estadística y econometría se verán en aplicaciones posteriores.

Sobre los datos utilizados:

  • Datos para el ejemplo : palmerpenguins (https://allisonhorst.github.io/palmerpenguins/)

Variación de la masa corporal media a lo largo de los años por especie

df %>%
  group_by(species, year) %>%
  summarize(N = n(),
            avg = mean(body_mass_g, na.rm = TRUE),
            SE = sd(body_mass_g, na.rm = TRUE) / sqrt(N),
            .groups = "drop") %>%
  ggplot(aes(x = year, y = avg)) +
  geom_ribbon(aes(ymin = avg - SE, ymax = avg + SE,
                  fill = species),
              alpha = 0.3) +
  geom_line(aes(color = species))

  return 'Island2'

  return 'Island3'
